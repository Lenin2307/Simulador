Etiquetas Semánticas

Son elementos en HTML5 que describen claramente el significado de su contenido, facilitando la interpretación tanto para el navegador como para el desarrollador.

Hoy en día, representan un estándar obligatorio para crear sitios web profesionales. Ayudan a los motores de búsqueda (SEO) a comprender la estructura y relevancia del contenido, además de ser vitales para la accesibilidad, permitiendo que las tecnologías de asistencia para personas con discapacidad lean el sitio de forma adecuada.

Lista de elementos semánticos de uso común

HTML5 Estándar
<header>

Contenedor para el contenido inicial de presentación, logotipos o el conjunto de enlaces de navegación principal.

<nav>

Define un bloque o sección especial destinado exclusivamente a contener enlaces de navegación interna o externa.

<main>

Engloba el contenido principal, único e importante de la página web. No debe repetirse en otros archivos.

<section>

Agrupa fragmentos de contenido que están estrechamente relacionados en un mismo tema.

<article>

Se utiliza para bloques de contenido independientes o autónomos (como noticias, posts de blogs o comentarios).

<aside>

Define secciones laterales que contienen información secundaria o complementaria respecto al flujo central.

<footer>

Pie de página del documento. Ideal para información del autor, avisos legales, enlaces de contacto y derechos de autor.
