Inicio: ¿Qué ocurre al ingresar una URL?

Cuando un usuario busca o escribe una dirección web (URL) en el navegador, empieza un proceso de comunicación entre cliente y servidor.

Es una dirección única que se utiliza para encontrar un recurso específico en la red de Internet; el navegador hace una consulta al sistema de Nombres de Dominio (DNS) para convertirla en el nombre de la página. Luego establece la conexión segura y el servidor la procesa y devuelve los archivos correspondientes.

Pasos Iniciales

  • 1 Usuario escribe una URL en la barra
  • 2 El navegador identifica el sitio mediante DNS
  • 3 Se prepara y envía una solicitud HTTP
