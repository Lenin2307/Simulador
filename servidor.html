Servidor Web: Procesamiento

Es el conjunto de operaciones que se ejecutan en la máquina central (servidor) al recibir una solicitud HTTP de un usuario.

El servidor HTTP analiza la URL solicitada y determina si el recurso es estático (como una imagen) o dinámico. Si la página es dinámica, el servidor interactúa con lenguajes de backend para ejecutar operaciones lógicas, validaciones de seguridad y consultas a bases de datos con el fin de generar el contenido antes de enviarlo de vuelta al navegador.

Cabeceras (Headers) HTTP Comunes

Metadata de Red
User-Agent

Identifica de forma clara el navegador, el sistema operativo o el cliente de software que está realizando la solicitud web.

Accept

Indica al servidor cuáles formatos de contenido (JSON, HTML, imágenes, etc.) es capaz de procesar y entender el cliente en la respuesta.

Content-Type

Determina el formato o tipo de medio MIME exacto del cuerpo del mensaje (por ejemplo, text/html o application/json).
